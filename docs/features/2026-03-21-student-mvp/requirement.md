# 学生管理系统 MVP - 需求文档

## 1. 项目概述

- **项目名称**: 学生管理系统 MVP
- **版本**: 1.0.0
- **描述**: 最简单的学生信息管理 MVP，只包含学生信息的 CRUD 功能
- **目标用户**: 学校管理员

## 2. 功能需求

### 2.1 学生列表
- 显示所有学生信息（分页，每页 10 条）
- 支持按姓名搜索
- 显示字段：ID、学号、姓名、性别、出生日期、联系电话

### 2.2 添加学生
- 表单字段：学号、姓名、性别、出生日期、联系电话、邮箱、地址
- 学号唯一性校验
- 必填字段：学号、姓名

### 2.3 编辑学生
- 预填现有数据
- 学号唯一性校验（排除自身）

### 2.4 删除学生
- 确认后删除
- 软删除或直接删除

### 2.5 查看详情
- 显示学生完整信息

## 3. 数据模型

### 3.1 Student 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| student_no | VARCHAR(20) UNIQUE | 学号（唯一） |
| name | VARCHAR(100) | 姓名 |
| gender | ENUM('男', '女') | 性别 |
| birth_date | DATE | 出生日期 |
| phone | VARCHAR(20) | 联系电话 |
| email | VARCHAR(100) | 邮箱 |
| address | VARCHAR(255) | 地址 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 4. API 接口设计

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/students | 获取学生列表（支持分页、搜索） |
| GET | /api/students/{id} | 获取单个学生详情 |
| POST | /api/students | 创建学生 |
| PUT | /api/students/{id} | 更新学生 |
| DELETE | /api/students/{id} | 删除学生 |

## 5. 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | 原生 PHP + MySQL |
| 前端 | Vue 3 + Vite |
| 数据库 | MySQL 8.0 |
