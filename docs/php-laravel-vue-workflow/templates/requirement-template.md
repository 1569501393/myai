## 3. API 接口设计

### 3.1 接口列表

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 获取列表 | GET | /api/features | 获取功能列表 |
| 创建 | POST | /api/features | 创建功能 |
| 更新 | PUT | /api/features/{id} | 更新功能 |
| 删除 | DELETE | /api/features/{id} | 删除功能 |

### 3.2 接口详情

#### GET /api/features

**请求参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码 |
| per_page | int | 否 | 每页数量 |
| status | string | 否 | 状态筛选 |

**响应示例**:
