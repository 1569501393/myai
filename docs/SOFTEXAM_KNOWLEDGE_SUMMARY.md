## 二、整合管理

### 2.1 核心过程

| 过程 | 输入 | 工具 | 输出 |
|------|------|------|------|
| 制定项目章程 | 项目工作说明书、商业论证 | 专家判断、会议 | 项目章程 |
| 制定项目管理计划 | 项目章程、协议 | 专家判断、头脑风暴 | 管理计划 |
| 指导与管理项目工作 | 管理计划、事业环境因素 | 专家判断、会议 | 可交付成果 |
| 管理项目知识 | 管理计划、可交付成果 | 专家判断、知识管理 | 经验教训登记册 |
| 监控项目工作 | 管理计划、协议、事业环境因素 | 专家判断、分析技术 | 变更请求 |

| 结束项目或阶段 | 管理计划、协议、可交付成果 | 专家判断、数据分析 | 最终产品、最终报告 |

### 3.2 WBS 记忆

```
WBS 创建原则：

100% 原则要遵守。
工作包是基础，
控制账户最顶头。

### 3.3 范围蔓延 vs 范围潜变

| 术语 | 英文 | 区别 |
|------|------|------|
| 范围蔓延 | Scope Creep | 未经批准的范围扩大 |
| 范围潜变 | Scope Creep (Gold Plating) | 客户提出新需求 |

### 3.4 记忆口诀

```
范围管理三步走：
收集需求是开头，
定义范围写说明，
WBS 分解到工作包。

控制范围防蔓延。

## 四、进度管理

### 4.1 核心过程

| 过程 | 说明 |
|------|------|
| 规划进度管理 | 制定进度管理计划 |
| 定义活动 | 识别分解后的活动 |
| 排列活动顺序 | 确定依赖关系 |
| 估算活动持续时间 | 估算所需时间 |
| 制定进度计划 | 分析进度安排 |
| 控制进度 | 监控进度变更 |

### 4.2 估算方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 类比估算 | 与类似项目比较 | 历史数据充分 |
| 参数估算 | 基于历史参数模型 | 参数可量化 |
| 三点估算 | (O+4M+P)/6 | 需要考虑风险 |
| 自下而上 | 从工作包向上汇总 | 详细信息可用 |

### 4.3 关键路径法记忆

```
关键路径是关键，
总时差为零是特点。
工期由它来决定，
压缩工期找关键。
自由时差是空档，
不影响后续工作的那点空。
```

### 4.4 网络图记忆

```
紧前关系图 PDM 两种符号：
方块节点 AON（活动在节点）
箭线节点 AOA（活动在箭线）。
单代号网络图，ES/EF/LS/LF 要记清。
最早从前往后算，最晚从后往前推。
总时差 = LS - ES = LF - EF。
```

### 4.5 进度压缩记忆

```
进度压缩两方法：
赶工Crashing：加钱不减范围，
              优先压缩关键路径上成本最低的活动。
快速跟进Fast Tracking：并行开展活动，
              增加风险和返工。
```

## 五、成本管理

### 5.1 核心过程

| 过程 | 说明 |
|------|------|
| 规划成本管理 | 制定成本管理计划 |
| 估算成本 | 估算所需资源成本 |
| 制定预算 | 汇总估算确定成本基准 |
| 控制成本 | 监控成本绩效 |

### 5.2 挣值管理 EVM 公式

```
┌─────────────────────────────────────────────────────────┐
│                     EVM 三兄弟                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  PV (计划值)     = 计划完成 × 预算单价                   │
│  EV (挣值)       = 实际完成 × 预算单价                   │
│  AC (实际成本)   = 实际完成 × 实际单价                   │
│                                                         │
│  CV (成本偏差)   = EV - AC     (>0 省钱)                │
│  SV (进度偏差)   = EV - PV     (>0 提前)                │
│                                                         │
│  CPI (成本绩效)  = EV / AC    (>1 省钱)                │
│  SPI (进度绩效)  = EV / PV    (>1 提前)                 │
│                                                         │
│  BAC = 总预算                                            │
│  EAC = BAC / CPI (典型偏差)                             │
│  EAC = AC + ETC  (非典型偏差)                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 5.3 记忆口诀

```
成本管理四步走：
估算成本是开头，
制定预算来汇总，
控制成本要监控。
挣值管理三参数，
PV EV AC 要分清。
CV SV 算偏差，
CPI SPI 看绩效。
```

## 九、风险管理

### 9.1 核心过程

| 过程 | 说明 |
|------|------|
| 规划风险管理 | 制定风险管理计划 |
| 识别风险 | 识别潜在风险 |
| 定性风险分析 | 评估概率和影响 |
| 定量风险分析 | 量化风险影响 |
| 规划风险应对 | 制定应对策略 |
| 实施风险应对 | 执行风险应对 |
| 监督风险 | 监控风险实施 |

### 9.2 风险应对策略

| 策略 | 威胁 | 机会 |
|------|------|------|
| 规避 | 消除威胁 | 消除机会 |
| 转移 | 转移给第三方 | 与他人分享 |
| 减轻 | 降低概率/影响 | 提升概率/影响 |
| 接受 | 承认但不处理 | 利用机会 |

### 9.3 决策树分析

```
决策树用EMV：
概率收益要相乘，
各路期望来相加，
选择最大是正道。
```

### 9.4 蒙特卡洛模拟

```
蒙特卡洛模拟：
多次模拟随机数，
概率分布来建模，
累计曲线得结论。
```

### 9.5 记忆口诀

```
风险管理七过程：
规划识别是开头，
定性与量来分析，
规划应对来制定，
实施应对来执行，
监督风险来控制。
风险策略四字诀：
规避转移减轻受。
威胁用前三种，
机会要受或利用。
```

## 十、采购管理

### 10.1 核心过程

| 过程 | 说明 |
|------|------|
| 规划采购管理 | 决定采购什么、如何采购 |
| 实施采购 | 获取卖方响应、选择卖方 |
| 控制采购 | 管理采购关系、监控绩效 |

### 10.2 合同类型

| 类型 | 风险 | 适用场景 |
|------|------|----------|
| 固定总价 FFP | 买方低 | 范围明确 |
| 总价加激励 FPIP | 中 | 绩效重要 |
| 成本加激励 CPIF | 中 | 难度大 |
| 成本加奖励 CPAF | 中 | 需激励 |
| 成本加固定费 CPFF | 卖方低 | 研究开发 |
| 工料合同 T&M | 双方 | 范围不明确 |

### 10.3 记忆口诀

```
采购管理三过程：
规划实施和控制。
合同类型要分清：
总价合同范围定，
成本合同助开发，
工料合同不明确。
```

## 十三、核心公式速查

```
┌─────────────────────────────────────────────────────────┐
│                     公式速查表                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【三点估算】                                            │
│  To = (O + 4M + P) / 6                                │
│  σ² = ((P - O) / 6)²                                  │
│                                                         │
│  【关键路径】                                            │
│  关键路径 = 总时差为零的活动路径                          │
│  项目总工期 = 关键路径长度                               │
│                                                         │
│  【挣值管理】                                            │
│  CV = EV - AC                                          │
│  SV = EV - PV                                          │
│  CPI = EV / AC                                         │
│  SPI = EV / PV                                         │
│  EAC = BAC / CPI (典型)                                │
│  EAC = AC + (BAC - EV) (非典型)                        │
│                                                         │
│  【沟通渠道】                                            │
│  N(N-1) / 2                                           │
│                                                         │
│  【决策树 EMV】                                          │
│  EMV = Σ(概率 × 收益)                                   │
│                                                         │
│  【投资回报】                                            │
│  ROI = (年收益 - 年成本) / 年成本 × 100%               │
│                                                         │
│  【期望货币价值】                                        │
│  EMV = Σ(结果i × 概率i)                                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
