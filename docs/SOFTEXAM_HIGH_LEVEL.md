# 软考高项备考指南

> 2026年5月考试 · 90天备考策略

## 概述

信息系统项目管理师（简称"高项"）是软考中难度较高的科目，考试内容涵盖：
- **综合知识**（选择题，75道）
- **案例分析**（3道大题）
- **论文**（2500字左右）

## 四、计算题专题

### 4.1 关键路径法 (CPM)

```python
# 关键路径计算
活动网络图:
A(0-2) → B(2-8) → D(8-12)
            ↓
        C(2-6) → E(6-12)

总工期 = 12天
关键路径 = A-B-D

# ES (最早开始) = 前置活动 EF
# EF (最早完成) = ES + 工期
# LS (最晚开始) = LF - 工期
# LF (最晚完成) = 后续活动 LS 的最小值
# 总时差 = LS - ES = LF - EF
# 自由时差 = 后续活动 ES - 当前活动 EF
```

### 4.2 挣值管理 (EVM)

```
PV (计划值)      = 计划完成的工作 × 预算单价
EV (挣值)        = 实际完成的工作 × 预算单价
AC (实际成本)    = 实际完成的工作 × 实际单价

CV (成本偏差)    = EV - AC
SV (进度偏差)    = EV - PV

CPI (成本绩效)   = EV / AC  (>1省钱,<1超支)
SPI (进度绩效)   = EV / PV  (>1提前,<1落后)

BAC (完工预算)   = 项目总预算
EAC (完工估算)   = BAC / CPI  或  AC + ETC

ETC (完工尚需)   = EAC - AC
VAC (完工偏差)   = BAC - EAC
```

### 4.3 决策树分析

```python
# 期望货币价值 (EMV)
决策点 A:
├── 方案1 (0.6): 收益100万 - 成本40万 = 60万
└── 方案2 (0.4): 收益150万 - 成本80万 = 70万

EMV(方案1) = 0.6 × 60 + 0.4 × (-10) = 32万
EMV(方案2) = 0.6 × 40 + 0.4 × 70 = 52万

结论: 选择方案2 (EMV更高)
```

```markdown

```
