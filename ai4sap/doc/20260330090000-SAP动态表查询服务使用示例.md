# SAP 动态表查询服务使用示例

## 服务端点

```
http://mysap.goodsap.cn:50400/sap/bc/zzjq?sap-client=400
```

## 认证方式

- Basic Auth: `U1170:hysoft888999`

## 使用示例

### 1. 查询 MARA 表 (物料主数据)

```bash
# 查询 MARA 表数据 (物料主数据)
curl -u U1170:hysoft888999 "http://mysap.goodsap.cn:50400/sap/bc/zzjq?sap-client=400&table=MARA"

# 带 limit 参数 (注意: 当前版本 limit 参数暂未生效)
curl -u U1170:hysoft888999 "http://mysap.goodsap.cn:50400/sap/bc/zzjq?sap-client=400&table=MARA&limit=5"
```

**响应示例** (返回 MARA 表物料数据):
```json
[
  {"MANDT":"400","MATNR":"000000000000000169","ERSDA":"2019-10-24","MTART":"HALB","MBRSH":"M","MATKL":"L003","MEINS":"ST",...},
  {"MANDT":"400","MATNR":"000000000000000173","ERSDA":"2019-10-24","MTART":"HALB","MBRSH":"M","MATKL":"L003","MEINS":"ST",...}
]
```

### 3. 查询 SPFLI 表 (航班时刻表)

```bash
# 查询 SPFLI 表
curl -u U1170:hysoft888999 "http://mysap.goodsap.cn:50400/sap/bc/zzjq?sap-client=400&table=SPFLI"
```

## 参数说明

| 参数 | 必需 | 说明 | 示例 |
|------|------|------|------|
| `sap-client` | ✅ | SAP 客户端 | `400` |
| `table` | ✅ | 表名 (必须大写) | `MARA`, `ZTJQ0003` |
| `limit` | ❌ | 返回行数 (暂未生效) | `5`, `10` |
