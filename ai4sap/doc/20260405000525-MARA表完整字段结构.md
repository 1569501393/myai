# MARA 表完整结构文档

## 概述

| 项目 | 值 |
|------|-----|
| **表名** | MARA |
| **描述** | General Material Data (一般物料数据) |
| **表类型** | TRANSPARENT (透明表) |
| **交付类** | A |
| **总字段数** | ~150+ (包含EMARA) |
| **主键** | MANDT + MATNR |

## 获取方法

```bash
# 获取 DDL 定义
curl -u "USER:PASS" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/ddic/tables/MARA/source/main" \
  -H "Accept: text/plain"

# 获取字段列表
curl -u "USER:PASS" \
  "http://mysap.goodsap.cn:50400/sap/bc/zzjq?table=MARA&rows=1" \
  -H "sap-client: 400"
```
