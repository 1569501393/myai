## 二、完整工作流程

### 阶段 1: 环境准备

```
1. 配置 SAP 连接 (config/sap_config.yaml)
2. 安装 Python 依赖 (requests, pyyaml)
3. 验证连接可用性
```

### 阶段 2: 源码获取流程

| 步骤 | 操作 | API 端点 | 状态码 |
|------|------|----------|--------|
| 1 | 获取程序源码 | `/sap/bc/adt/programs/programs/{name}/source/main` | 200 |
| 2 | 获取包含文件 | `/sap/bc/adt/programs/includes/{name}/source/main` | 200 |
| 3 | 获取类源码 | `/sap/bc/adt/oo/classes/{name}/source/main` | 200 |
| 4 | 获取接口源码 | `/sap/bc/adt/oo/interfaces/{name}/source/main` | 200 |
| 5 | 获取表定义 | `/sap/bc/adt/ddic/tables/{name}/source/main` | 200 |

### 阶段 3: 数据获取流程

```
1. 确定目标表和字段
2. 编写 ABAP 程序查询数据
3. 执行程序获取结果
4. 解析输出数据
```

### 阶段 4: 分析输出流程

```
1. 解析源码结构
2. 提取关键信息 (变量、FORM、类、方法)
3. 生成流程图 (Mermaid)
4. 输出标准化文档
```

## 四、最佳实践

### 4.1 配置管理

```yaml
# config/sap_config.yaml
SAP_CONFIG:
  USER: ${SAP_USER}        # 使用环境变量
  PASSWD: ${SAP_PASSWD}    # 使用环境变量
  CLIENT: "400"
  LANG: ZH

ADT_CONFIG:
  ADT_PROTOCOL: https      # 生产环境使用 HTTPS
  ADT_HOST: sap.company.com
  ADT_PORT: 443
  BASE_URL: https://sap.company.com:443
```

### 4.2 安全最佳实践

| 实践 | 说明 |

## 五、SAP ADT REST API 已验证端点

### 5.1 源码获取 (100% 可用)

| 对象类型 | API 端点 | 说明 |
|----------|----------|------|
| PROG | `/sap/bc/adt/programs/programs/{name}/source/main` | 报表程序 |
| INCLUDE | `/sap/bc/adt/programs/includes/{name}/source/main` | 包含文件 |
| CLASS | `/sap/bc/adt/oo/classes/{name}/source/main` | ABAP 类 |
| INTF | `/sap/bc/adt/oo/interfaces/{name}/source/main` | 接口 |
| TABL | `/sap/bc/adt/ddic/tables/{name}/source/main` | 数据库表 |

### 5.2 请求头模板

```python
headers = {
    'Authorization': f'Basic {base64.b64encode(f"{user}:{passwd}".encode()).decode()}',
    'Accept': 'text/plain',
    'sap-client': '400',
    'sap-language': 'ZH'
}
```

### 5.3 错误处理

| 状态码 | 原因 | 处理 |
|--------|------|------|
| 200 | 成功 | 正常处理 |
| 401 | 认证失败 | 检查用户名密码 |
| 404 | 对象不存在 | 确认对象名称 |
| 406 | Accept 不支持 | 修改 Accept 头 |
| 500 | 服务器错误 | 重试或联系管理员 |

## 六、Python 代码模板

### 6.1 获取单个对象源码

```python
from sap_connector import SAPConnector, load_config

config = load_config()
conn = SAPConnector(config)

result = conn.get_source('PROG', 'ZJQR0000')
if 'source' in result:
    print(result['source'])
else:
    print(f"Error: {result.get('error')}")
```

### 6.2 获取完整对象 (含包含文件)

```python
result = conn.get_full_object('PROG', 'ZJQR0000')
print(f"Source: {result.get('source_length')} chars")
print(f"Includes: {result.get('includes')}")
print(f"Text Symbols: {result.get('text_symbols')}")
```

### 6.3 批量获取

```python
objects = [
    ('PROG', 'ZJQR0000'),
    ('INCLUDE', 'ZJQR0000_FRM'),
    ('CLASS', 'ZCL_TEST'),
]

for obj_type, obj_name in objects:
    result = conn.get_source(obj_type, obj_name)
    filename = f"output/{obj_name}.src"
    with open(filename, 'w') as f:
        f.write(result.get('source', ''))
```

## 七、常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 401 Unauthorized | 密码错误 | 检查 config/sap_config.yaml |
| 404 Not Found | 对象不存在 | 确认对象名大小写 |
| 406 Not Acceptable | Accept 头错误 | 使用 text/plain |
| 连接超时 | 网络问题 | 检查网络和防火墙 |
| 中文乱码 | 编码问题 | 使用 UTF-8 编码 |
