## 二、文本符号 (Text Symbols)

### 2.1 D020T - 屏幕文本表

**用途**: 存储选择屏幕和普通屏幕的描述文本

| 字段 | 类型 | 说明 |
|------|------|------|
| MANDT | MANDT | 客户端 |
| PROG | CHAR(40) | 程序名称 |
| DYNR | NUMC(4) | 屏幕号 |
| LANG | LANG | 语言代码 |
| DTXT | CHAR(70) | 屏幕描述文本 |

**查询示例**:
```sql
-- 查询特定程序的屏幕文本
SELECT prog, dynr, lang, dtxt 
FROM d020t 
WHERE prog = 'ZJQR000'
  AND lang = '1'
```

**数据样例**:
| PROG | DYNR | LANG | DTXT |
|------|------|------|-------|
| ZJQR000 | 1000 | 1 | SEL_SCREEN 1000 INFO |
| ZMM578 | 9000 | 1 | 工厂成品组别入库统计报表 |

## 三、选择文本 (Selection Texts)

### 3.1 D021S - 选择屏幕字段文本表

**用途**: 存储选择屏幕上参数和选择选项的文本标签

| 字段 | 类型 | 说明 |
|------|------|------|
| MANDT | MANDT | 客户端 |
| PROG | CHAR(40) | 程序名称 |
| DNUM | CHAR(4) | 选择屏幕号 |
| NAME | CHAR(30) | 参数/选择选项名称 |
| TEXT | CHAR(55) | 文本标签 |
| LANGU | LANG | 语言代码 |

**查询示例**:
```sql
-- 查询程序的选择文本
SELECT prog, dnum, name, text 
FROM d021s 
WHERE prog = 'ZJQR000'
```

### 3.2 TRDIRT - 程序文本表

**用途**: 存储程序的描述性文本（程序属性中的"简短描述"）

| 字段 | 类型 | 说明 |
|------|------|------|
| MANDT | MANDT | 客户端 |
| NAME | CHAR(40) | 程序名称 |
| SPRSL | LANG | 语言代码 |
| TEXT | CHAR(70) | 程序描述 |

**查询示例**:
```sql
-- 查询程序文本
SELECT name, sprsl, text 
FROM trdirt 
WHERE name = 'ZJQR000'
```

**数据样例**:
| NAME | SPRSL | TEXT |
|------|-------|------|
| ZJQR000 | 1 | 测试 |
| ZMM578 | 1 | 工厂成品组别入库统计报表 |

| MANDT | MANDT | 客户端 |

**查询示例**:
