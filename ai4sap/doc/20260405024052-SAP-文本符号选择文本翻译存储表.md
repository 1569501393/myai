## 一、概述

SAP ABAP 开发对象中的文本、文本符号和翻译信息存储在多个表中。本文档详细说明各类型文本的存储位置和查询方法。

## 二、文本类型分类表

### 2.1 总览

| 文本类型 | 存储表 | 说明 |
|----------|--------|------|
| **文本符号** | D020T, D021S | 选择屏幕文本、参数标签 |
| **程序文本** | TRDIRT | 程序描述文本 |
| **消息文本** | T100 | 消息类中的消息文本 |
| **函数模块文本** | TFTIT | 函数模块描述 |

## 三、详细表说明

### 3.1 D020T - 屏幕文本 (Screen Texts)

**用途**: 存储选择屏幕和普通屏幕的描述文本

| 字段 | 类型 | 说明 |
|------|------|------|
| PROG | CHAR(40) | 程序名称 |
| DYNR | NUMC(4) | 屏幕号 |
| LANG | LANG | 语言代码 |
| DTXT | CHAR(70) | 屏幕描述文本 |

**查询示例**:
```sql
SELECT prog, dynr, dtxt 
FROM d020t 
WHERE prog = 'ZJQR000'
```

---

### 3.2 D021S - 选择屏幕字段文本 (Selection Screen Field Texts)

**用途**: 存储选择屏幕上参数和选择选项的文本标签

| 字段 | 类型 | 说明 |
|------|------|------|
| PROG | CHAR(40) | 程序名称 |
| DNUM | CHAR(4) | 选择屏幕号 |
| NAME | CHAR(30) | 参数/选择选项名称 |
| TEXT | CHAR(55) | 文本标签 |
| LANGU | LANG | 语言 |

**查询示例**:
```sql
SELECT prog, dnum, name, text 
FROM d021s 
WHERE prog = 'ZJQR000'
```

---

### 3.3 TRDIRT - 程序文本 (Program Texts)

**用途**: 存储程序的描述性文本（程序属性中的"简短描述"）

| 字段 | 类型 | 说明 |
|------|------|------|
| NAME | CHAR(40) | 程序名称 |
| SPRSL | LANG | 语言代码 |
| TEXT | CHAR(70) | 程序描述 |

**查询示例**:
```sql
SELECT name, sprsl, text 
FROM trdirt 
WHERE name = 'ZJQR000'
```

**数据样例**:
```
NAME:    ZJQR000
SPRSL:   1 (中文)
TEXT:    测试
```

---

### 3.4 T100 - 消息文本 (Message Texts)

**用途**: 存储消息类的消息文本

| 字段 | 类型 | 说明 |
|------|------|------|
| SPRSL | LANG | 语言代码 |
| ARBGB | CHAR(20) | 消息类 |
| MSGNR | CHAR(3) | 消息号 |
| TEXT | CHAR(73) | 消息文本 |

**查询示例**:
```sql
SELECT sprsl, arbgb, msgnr, text 
FROM t100 
WHERE arbgb = 'ZJQR'
```

---

### 3.5 TFTIT - 函数模块文本 (Function Module Texts)

**用途**: 存储函数模块的简短文本

| 字段 | 类型 | 说明 |
|------|------|------|
| FUNCNAME | CHAR(30) | 函数模块名称 |
| SPRAS | LANG | 语言 |
| STEXT | CHAR(60) | 函数描述 |

**查询示例**:
```sql
SELECT funcname, stext 
FROM tftit 
WHERE funcname LIKE 'ZJQR%'
```

---

### 3.6 STXH / STXL - 长文本 (Long Texts)

**用途**: 存储各种长文本，如物料描述、文档等

**STXH (长文本头表)**:
| 字段 | 类型 | 说明 |
|------|------|------|
| MANDT | MANDT | 客户端 |
| TDOBJECT | CHAR(10) | 对象类型 (如 MARA, KNA1) |
| TDNAME | CHAR(70) | 对象名称 (如物料号) |
| TDID | CHAR(4) | 文本ID (如 BEST, KOPF) |
| TDSPRAS | LANG | 语言 |
| TDTITLE | CHAR(70) | 标题 |
| TXTLINES | NUMC(5) | 文本行数 |

**STXL (长文本行表)**:
| 字段 | 类型 | 说明 |
|------|------|------|
| MANDT | MANDT | 客户端 |
| SRTF2 | NUMC(5) | 行号 |
| TDOBJECT | CHAR(10) | 对象类型 |
| TDNAME | CHAR(70) | 对象名称 |
| TDID | CHAR(4) | 文本ID |
| TDSPRAS | LANG | 语言 |

**查询示例**:
```sql
-- 查询物料长文本
SELECT * FROM stxh 
WHERE tdobject = 'MARA' 
  AND tdspras = '1'
```

### 3.7 T002 - 语言 (Languages)

| 字段 | 类型 | 说明 |

## 五、文本符号获取方法对比

### 5.1 ADT REST API 方式 (推荐)

```bash

  "http://mysap.goodsap.cn:50400/sap/bc/adt/textelements/programs/zjqr000" \
  -H "Accept: application/vnd.sap.adt.textelements.symbols+xml"
```

```bash
#!/bin/bash
# SAP 文本符号和翻译查询脚本

HOST="mysap.goodsap.cn"
PORT="50400"
CLIENT="400"
USER="U1170"
PASS="hysoft888999"

# 获取CSRF Token

  local sql="$1"
  curl -s -b /tmp/sap_cookies.txt \
    "http://${HOST}:${PORT}/sap/bc/adt/datapreview/freestyle?rowNumber=100" \
    -H "Content-Type: text/plain" \

echo ""
echo "=== 查询屏幕文本 (D020T) ==="
query_sap "SELECT prog, dynr, dtxt FROM d020t WHERE prog = '$1'"
```
