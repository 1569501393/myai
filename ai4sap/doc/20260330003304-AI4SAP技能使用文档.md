# AI4SAP技能使用文档

## 概述

AI4SAP技能是一个用于连接SAP系统、分析源码和表数据的Python REST API集成工具。专为需要与SAP系统交互、获取程序源码、表数据的用户设计。

## 环境要求

- Python 3.7+
- requests 库
- pyyaml 库

## 快速开始

### 1. 安装依赖

```bash
pip install requests pyyaml
```

### 2. 配置连接

创建配置文件 `sap_config.yaml`:

```yaml
SAP_CONFIG:
  USER: "您的用户名"
  PASSWD: "您的密码"
  CLIENT: "400"
  LANG: "ZH"

ADT_CONFIG:
  BASE_URL: "http://您的SAP服务器:端口"
```

### 3. 基本使用

```python
from sap_connector import SAPConnector
import yaml

# 加载配置
config = yaml.safe_load(open("sap_config.yaml"))

# 创建连接器
conn = SAPConnector(config)

# 获取程序源码
result = conn.get_full_object("PROG", "ZJQR0000")
print(result)

# 获取表数据
table_result = conn.get_table_data("MARA", rows=10)
print(table_result)
```

## 功能列表

### 1. 获取SAP对象源码

| 方法 | 说明 |
|------|------|
| `get_source(type, name)` | 获取指定对象的源码 |
| `get_full_object(type, name)` | 获取完整对象信息（含includes、text symbols） |
| `get_includes(program)` | 获取程序包含的INCLUDE文件列表 |
| `get_text_symbols(name, type)` | 提取文本符号（TEXT-XXX） |

### 2. 获取表数据

| 方法 | 说明 |
|------|------|
| `get_table_data(table, rows)` | 通过自定义端点获取表数据 |

### 3. 搜索SAP对象

| 方法 | 说明 |
|------|------|
| `search_objects(query, types)` | 按关键字搜索SAP对象 |
| `get_object_info(type, name)` | 获取对象元数据 |

## 支持的对象类型

- `PROG` - 程序
- `INCLUDE` - Include文件
- `CLASS` - 类
- `DOMAIN` - 域
- `DTEL` - 数据元素
- `TABL` - 表
- `VIEW` - 视图

## 具体使用场景和提示词

### 场景1：获取SAP程序源码

**提示词**：
```
获取 ZJQR0000 程序的完整源码，包括所有INCLUDE文件
```

**对应代码**：
```python
# 获取主程序
main_result = conn.get_full_object("PROG", "ZJQR0000")
print(f"主程序源码长度: {main_result['source_length']}")

# 获取所有INCLUDE文件
includes = main_result.get("includes", [])
for inc in includes:
    inc_result = conn.get_source("INCLUDE", inc)
    print(f"INCLUDE {inc}: {inc_result.get('length', 0)} 字符")
```

### 场景2：获取表结构和数据

**提示词**：

### 场景7：分析表数据并导出

**提示词**：
```
获取 MARA 表100条数据，分析物料类型分布，导出JSON文件
```

**对应代码**：
```python
import json
from pathlib import Path

# 获取数据
result = conn.get_table_data("MARA", rows=100)
data = result.get("data", [])

# 分析物料类型分布
mtart_count = {}
for row in data:
    mtart = row.get("MTART", "UNKNOWN")
    mtart_count[mtart] = mtart_count.get(mtart, 0) + 1

print(f"物料类型分布: {mtart_count}")

# 保存到文件
output = Path("output")
output.mkdir(exist_ok=True)
(output / "mara_analysis.json").write_text(
    json.dumps({"data": data, "analysis": mtart_count}, ensure_ascii=False, indent=2)
)
print("已保存到 output/mara_analysis.json")
```

### 场景10：完整项目分析

**提示词**：
```
对 ZJQR0000 项目进行全面分析：源码、INCLUDE、文本符号、调用关系
```

**对应代码**：
```python
import json

# 全面获取项目信息
project_name = "ZJQR0000"
result = conn.get_full_object("PROG", project_name)

# 构建分析报告
analysis = {
    "project_name": project_name,
    "source_length": result.get("source_length", 0),
    "includes": result.get("includes", []),
    "text_symbols": result.get("text_symbols", []),
    "has_error": "error" in result
}

# 获取每个INCLUDE的内容摘要
for inc in result.get("includes", [])[:5]:
    inc_result = conn.get_source("INCLUDE", inc)
    analysis[f"include_{inc}"] = {
        "length": inc_result.get("length", 0),
        "available": "source" in inc_result
    }

print(json.dumps(analysis, indent=2, ensure_ascii=False))
```

## SAP端点说明

### 标准ADT端点

| 端点 | 说明 |
|------|------|
| `/sap/bc/adt/programs/programs/{name}/source/main` | 程序源码 |
| `/sap/bc/adt/oo/classes/{name}/source/main` | 类源码 |
| `/sap/bc/adt/ddic/tables/{name}/definition` | 表结构 |
| `/sap/bc/adt/ddic/tables/{name}/data` | 表数据 |

### 自定义端点

| 端点 | 说明 |
|------|------|
| `/sap/bc/zzjq?table=xxx&limit=10` | 动态表查询 |

## 错误处理

```python
result = conn.get_source("PROG", "ZJQR0000")
if "error" in result:
    print(f"错误: {result['error']}")
else:
    print(f"成功获取源码，长度: {result['length']}")
```

## 常见问题

### Q: 认证失败怎么办？
A: 检查用户名密码是否正确，确认SAP用户有ADT权限

### Q: 返回404怎么办？
A: 确认对象名称正确，检查ADT服务是否启用

### Q: 表数据获取不到？
A: 某些系统未启用标准ADT表数据端点，可尝试自定义端点 `/sap/bc/zzjq`
