## 四、发现的问题

### 4.1 文本符号端点不可用

`/sap/bc/adt/textelements/*` 端点返回 404 错误，原因是相关 SICF 服务未激活。

**解决方案**: 在 SICF 事务码中激活 `/sap/bc/adt/textelements` 服务。

### 4.2 局部类型端点错误

当类没有局部类型时，访问 `/includes/localtypes` 会返回错误。

**解决方案**: 先检查类是否有局部类型，或捕获异常处理。

## 五、URL 编码说明

| 字符 | 编码 | 示例 |
|------|------|------|
| `/` | `%2f` | `/foo/bar` → `%2ffoo%2fbar` |
| `,` | `%2c` | `KAD_52,AT520019` → `KAD_52%2cAT520019` |
| 空格 | `%20` | `    rq` → `%20%20%20%20rq` |

## 六、补充说明

### 6.1 类包含类型

从类元数据响应中可以看到以下包含类型：
- `definitions` - 类定义（局部类型/接口）
- `implementations` - 实现部分
- `macros` - 宏定义
- `testclasses` - 测试类
- `main` - 主实现

### 6.2 Class Pool vs Class Name

在 TRDIR 表中：
- **Class Pool**: `ZCL_ABAP_DATABASE_CREATE=============CP`
- **Class Name**: `ZCL_ABAP_DATABASE_CREATE`

ADT API 使用类名，不是类池名。
