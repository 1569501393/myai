## 可用端点总结

### ✅ 完全可用的端点 (6个)

| 端点 | 说明 | 备注 |
|------|------|------|
| `/sap/bc/adt/programs/programs/{name}/source/main` | 获取程序源码 | text/plain |
| `/sap/bc/adt/programs/includes/{name}/source/main` | 获取Include源码 | text/plain |
| `/sap/bc/adt/oo/classes/{name}/source/main` | 获取类源码 | text/plain |
| `/sap/bc/adt/ddic/tables/{name}/source/main` | 获取表源码(DDL) | text/plain |

### ⚠️ 需要的替代方案

| 需要的功能 | 当前状态 | 替代方案 |
|-----------|----------|----------|
| 表数据 | ❌ `/sap/bc/adt/ddic/tables/{name}/data` 不可用 | 使用 `/sap/bc/zzjq?table={name}&rows={n}` |
| 表定义 | ❌ `/sap/bc/adt/ddic/tables/{name}/definition` 不可用 | 使用 `/sap/bc/zzjq?table={name}&rows=1` 获取字段 |
| 对象信息 | ❌ 406错误 | 使用 `/sap/bc/zzjq` 端点 |
| 数据预览 | ❌ `/sap/bc/adt/datapreview` 不可用 | 这是S/4HANA专用端点 |

---

## 系统类型判断

根据测试结果，当前SAP系统特征：

1. **支持源码获取** (`/source/main` 端点返回200)
2. **不支持DDIC元数据** (表/视图/域信息返回406)
3. **不支持数据预览** (`/sap/bc/adt/datapreview` 返回404)
4. **无ATC/代码检查权限** (返回403)
5. **大部分S/4HANA专用端点不可用**

**结论**: 该系统为 **SAP ECC** 或旧版NetWeaver系统，不是S/4HANA。

---

## 建议

1. **对于源码获取**: 使用 `/sap/bc/adt/{type}/{name}/source/main` 端点
2. **对于表数据**: 使用自定义端点 `/sap/bc/zzjq?table={name}&rows={n}`
3. **对于表结构**: 暂无标准ADT解决方案，需使用RFC或自定义开发
4. **需要升级**: 如需完整ADT支持，建议升级到S/4HANA或最新NetWeaver版本
