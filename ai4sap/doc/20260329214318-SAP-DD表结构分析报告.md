## 1. SAP 字典表概述

SAP ABAP 字典 (Data Dictionary) 是存储所有元数据的核心组件。以下是主要的 DD 表：

| 表名 | 用途 | 说明 |
|------|------|------|
| DD01L | 域 (Domain) | 存储域定义 |
| DD04L | 数据元素 (Data Element) | 存储数据元素定义 |
| DD02L | 表 (Table) | 存储透明表定义 |
| DD03L | 字段 (Table Field) | 存储表字段定义 |
| DD07L | 域值范围 (Value Range) | 存储域的固定值 |
| DD07T | 域值文本 (Value Text) | 存储域值描述文本 |

## 8. 获取域值的推荐方法

### 8.1 ADT REST API (推荐)

```bash
# 获取域定义 (包含固定值)
curl -u USER:PASS \
  "http://sap-host:port/sap/bc/adt/ddic/domains/BU_SEX"
```

### 8.2 直接 SQL 查询

```sql
-- 查询特定域的值范围 (需要知道具体的语言和客户端)
SELECT DOMNAME, VALPOS, DOMVALUE_L, DOMVALUE_H
FROM DD07L
WHERE DOMNAME = 'BU_SEX'
  AND DDLANGUAGE = '1'
  AND AS4LOCAL = 'A';

-- 查询值文本
SELECT DOMNAME, DDLANGUAGE, VALPOS, DDTEXT
FROM DD07T
WHERE DOMNAME = 'BU_SEX'
  AND DDLANGUAGE = '1'
  AND AS4LOCAL = 'A';
```

## 11. 结论

1. **DD01L (域)**: 存储域的基本信息，如数据类型、长度等
2. **DD04L (数据元素)**: 存储数据元素及其引用的域
3. **DD02L (表)**: 存储透明表的定义信息
4. **DD03L (字段)**: 存储表的字段定义
5. **DD07L/DD07T (值范围/文本)**: 存储域的固定值及描述

对于获取域的 Value Range，建议使用 **ADT REST API** (如 `/sap/bc/adt/ddic/domains/{domain_name}`) 而不是直接查询 DD07L/DD07T，因为：
- ADT API 返回完整的域定义，包括固定值
- DD07L/DD07T 可能因客户端/语言设置不同而数据不同
- ADT API 更加可靠和标准化
