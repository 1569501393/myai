## 2. API 使用示例

### 2.1 基本查询

```bash
# 查询 MARA 表前 5 条记录
/sap/bc/zzjq?table=MARA&rows=5
```

### 2.2 指定字段

```bash
# 查询 MARA 表的 MATNR 和 ERSDA 字段
/sap/bc/zzjq?table=MARA&fields=MATNR,ERSDA&rows=10
```

### 2.3 带 WHERE 条件

```bash
# 查询 KNA1 表中 LAND1='CN' 的记录
/sap/bc/zzjq?table=KNA1&where=LAND1='CN'&rows=20
```

### 2.4 带排序

```bash
# 查询 MARA 表，按 MATNR 升序
/sap/bc/zzjq?table=MARA&orderby=MATNR%20ASC&rows=10

# 按创建日期降序
/sap/bc/zzjq?table=MARA&orderby=ERSDA%20DESC&rows=10
```

### 2.5 完整示例

```bash
# 完整查询示例
/sap/bc/zzjq?table=KNA1&fields=KUNNR,NAME1,LAND1&where=LAND1='CN'&orderby=KUNNR&rows=20
```


### 5.2 核心代码片段

```abap
" 构建动态 SELECT
METHOD build_dynamic_select.
  
  " 字段列表处理
  IF iv_fields IS INITIAL.
    lv_fields = '*'.
  ELSE.
    REPLACE ALL OCCURRENCES OF ' ' IN iv_fields WITH ''.
    lv_fields = iv_fields.
  ENDIF.

  " 构建 SELECT 语句
  ev_select = |SELECT { lv_fields } FROM { iv_table }|.

  " WHERE 条件
  IF iv_where IS NOT INITIAL.
    ev_select = ev_select && | WHERE { iv_where }|.
  ENDIF.

  " ORDER BY
  IF iv_orderby IS NOT INITIAL.
    ev_select = ev_select && | ORDER BY { iv_orderby }|.
  ENDIF.

  " 行数限制
  IF iv_rows > 0.
    ev_select = ev_select && | UP TO { iv_rows } ROWS |.
  ENDIF.

  ev_select = ev_select && | INTO TABLE @DATA(lt_result)|.

ENDMETHOD.
```

## 6. 安全考虑

### 6.1 SQL 注入防护

- 验证 WHERE 条件中的关键词
- 禁止危险关键字 (DROP, DELETE, UPDATE, INSERT)
- 建议前端使用参数化查询

### 6.2 访问控制

- 当前版本无权限控制
- 建议生产环境添加:
  - 用户认证
  - IP 白名单
  - 表访问白名单

### 6.3 性能限制

- 最大返回 1000 条记录
- 防止全表扫描
