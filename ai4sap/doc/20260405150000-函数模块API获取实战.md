## 三、Include 命名规则

### 3.1 命名规范

| 类型 | 命名规则 | 示例 |
|------|----------|------|
| 函数组主程序 | `SAPL{group}` | `SAPLZFG_KF036` |
| TOP Include | `L{group}TOP` | `LZFG_KF036TOP` |
| 函数模块 Include | `L{num}{group}` | `LZFG_KF036U01` |
| 通用 Include | `L{group}UXX` | `LZFG_KF036UXX` |
| 特殊 Include | `L{group}$XX` | `LZFG_KF036$01` |
| 函数池 | `FUNCTION-POOL {group}` | `FUNCTION-POOL ZFG_KF036` |

### 3.2 Include 类型 (SUBC)

| 值 | 类型 | 说明 |
|----|------|------|
| `F` | Function Group | 函数组主程序 |
| `I` | Include | Include 程序 |
| `1` | Module Pool | 模块池 |
| `M` | Method | 方法 |

## 六、TFDIR 表结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `FUNCNAME` | CHAR 30 | 函数模块名称 |
| `PNAME` | CHAR 40 | 函数组程序池名称 (如 `SAPLZFG_KF036`) |
| `INCLUDE` | CHAR 4 | Include 编号 (如 `01`, `02`) |
| `FREEDATE` | DATS | 释放日期 |
| `APPL` | CHAR 1 | 应用组件 |
| `FMODE` | CHAR 1 | 功能模式 |
| `HOST` | CHAR 32 | 宿主系统 |
| `RFCSCOPE` | CHAR 1 | RFC 范围 |
| `RFCVERS` | CHAR 1 | RFC 版本 |

## 八、总结

### 8.1 核心发现

1. **Include 命名规则**: `L{编号}{函数组名(不含SAPL)}`
   - 例: `LZFG_KF036U01` (编号01, 函数组 ZFG_KF036)

2. **函数组 API**: 使用小写函数组名
   - 例: `/sap/bc/adt/functions/groups/zfg_kf036/source/main`

3. **Include API**: 直接使用完整 Include 名
   - 例: `/sap/bc/adt/programs/includes/LZFG_KF036U01/source/main`

### 8.2 获取流程

```
TFDIR 查询 
    ↓
获取 PNAME (如 SAPLZFG_KF036) 
    ↓
提取函数组名 (如 zfg_kf036)
    ↓
获取函数组源码: /sap/bc/adt/functions/groups/zfg_kf036/source/main
    ↓
获取 Include 源码: /sap/bc/adt/programs/includes/{Include名}/source/main
```
