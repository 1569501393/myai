## 系统类型判断

基于测试结果，当前SAP系统特征:

1. ✅ 支持源码获取 (`/source/main` 端点)
2. ❌ 不支持DDIC元数据 (表/视图/域/数据元素)
3. ❌ 不支持数据预览 (`/sap/bc/adt/datapreview`)
4. ❌ 不支持ATC/代码检查 (权限不足)
5. ❌ 不支持传输管理
6. ❌ 不支持对象搜索
7. ❌ 不支持Where Used分析

**结论**: 该系统为 **SAP ECC** 或旧版NetWeaver系统，**不是S/4HANA**。

## 替代方案

| 需要的功能 | 标准ADT端点 | 替代方案 |
|-----------|-------------|----------|
| 表数据 | `/sap/bc/adt/ddic/tables/{name}/data` | `/sap/bc/zzjq?table={name}&rows=10` |
| 表定义 | `/sap/bc/adt/ddic/tables/{name}/definition` | `/sap/bc/zzjq?table={name}` |
| 对象搜索 | `/sap/bc/adt/repository/objectsearch` | SE16/SE11手动查询 |
| Where Used | `/sap/bc/adt/navigation/whereused` | SE80/SE84事务码 |

---

## 建议

1. **对于源码获取**: 使用 `/sap/bc/adt/{type}/{name}/source/main` 端点 ✅
2. **对于表数据**: 使用自定义端点 `/sap/bc/zzjq?table={name}` ✅
3. **对于表结构**: 暂无标准ADT解决方案，需使用RFC或自定义开发
4. **需要完整功能**: 建议升级到 S/4HANA 或最新 NetWeaver 版本
