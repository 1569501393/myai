# SAP /sap/bc/zzjq 端点资源清单

## 概述

`/sap/bc/zzjq` 是 SAP 系统上的一个自定义 REST API 端点，用于获取表数据。该端点通过 RFC_READ_TABLE 功能模块实现，支持基本的表数据查询。

**测试系统**: mysap.goodsap.cn:50400
**客户端**: 400 (S4HCLNT400)
**用户**: U1170
**测试日期**: 2026-04-04

---

## 端点基础信息

| 项目 | 值 |
|------|-----|
| **端点路径** | `/sap/bc/zzjq` |
| **HTTP方法** | GET, POST |
| **认证方式** | Basic Auth |
| **响应格式** | JSON |
| **内容类型** | `application/json; charset=utf-8` |

---

## 支持的参数

### 1. table (必需)

**说明**: 表名

**示例**:
```
table=mara
table=MARA
table=T000
table=KNA1
```

**测试结果**: ✅ 支持

### 2. rows (可选)

**说明**: 请求返回行数

**示例**:
```
rows=3
rows=10
rows=100
```

**⚠️ 注意**: 实际测试发现，无论 rows 参数设置为何值，API 固定返回 **10 条记录**。

**测试结果**: ⚠️ 参数被接受但实际不生效

### 3. sql (可选)

**说明**: 直接传递 SQL 语句

**示例**:
```
sql=SELECT * FROM MARA UP TO 3 ROWS
```

**测试结果**: ✅ 支持 (返回1条记录)

### 4. query (可选)

**说明**: SQL 查询语句 (sql 的别名)

**测试结果**: ✅ 支持

## 请求示例

### 基本查询

```bash
# 获取MARA表数据
curl -u "U1170:hysoft888999" \
  "http://mysap.goodsap.cn:50400/sap/bc/zzjq?table=mara" \
  -H "sap-client: 400"
```

**响应** (固定返回10条):
```json
[

  {"MANDT":"400","MATNR":"000000000000000173","ERSDA":"2019-10-24",...},
  ...
```

```bash

  -H "sap-client: 400" \
```

## 已知限制

1. **固定返回10条**: 无论 `rows` 参数设置为何值，API 固定返回 10 条记录
2. **不支持字段过滤**: `fields` 参数不生效
3. **不支持 WHERE 条件**: 不能按条件筛选
4. **不支持排序**: 不能对结果排序
5. **不支持分页**: 没有 offset/limit 支持
6. **不支持表结构**: 不能获取表定义

## Python 使用示例

```python
import requests
from requests.auth import HTTPBasicAuth

SAP_HOST = "mysap.goodsap.cn:50400"
AUTH = HTTPBasicAuth("U1170", "hysoft888999")
CLIENT = "400"

def get_table_data(table_name, rows=10):
    """获取表数据 (固定返回10条)"""
    url = f"http://{SAP_HOST}/sap/bc/zzjq"
    
    response = requests.get(
        url,
        auth=AUTH,
        params={"table": table_name, "rows": rows},
        headers={"sap-client": CLIENT}
    )
    
    if response.status_code == 200:
        return response.json()
    else:
        return {"error": f"HTTP {response.status_code}", "detail": response.text}

# 使用示例
data = get_table_data("MARA")
for row in data:
    print(row["MATNR"], row["MTART"])
```

## 结论

`/sap/bc/zzjq` 端点是一个简单的表数据查询接口，适合:
- 快速获取表数据预览
- 开发测试环境
- 数据探索

**局限性**:
- 固定返回10条，不适合大批量数据
- 不支持条件筛选、排序、分页

**建议**: 对于复杂查询需求，考虑使用 RFC_READ_TABLE 直接调用或开发自定义功能模块。
