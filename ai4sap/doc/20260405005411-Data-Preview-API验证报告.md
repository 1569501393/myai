## 关键发现

### 1. CSRF Token 获取

**端点**: `GET /sap/bc/adt/discovery`

```bash
curl -u "USER:PASS" \
  "http://host:port/sap/bc/adt/discovery" \
  -H "Accept: application/atomsvc+xml" \
  -H "sap-client: 400" \
  -H "x-csrf-token: Fetch"
```

**响应头**: `X-CSRF-Token: <token>`

### 2. freestyle 端点 (SQL 查询)

**端点**: `POST /sap/bc/adt/datapreview/freestyle`

**请求头**:
- `Accept: application/vnd.sap.adt.datapreview.table.v1+xml`
- `Content-Type: text/plain`
- `X-CSRF-Token: <token>`

**请求参数**:
- `rowNumber` - 返回行数

**请求体**: 纯文本 SQL 语句

**示例**:
```sql
SELECT matnr, mtart, matkl FROM mara UP TO 10 ROWS
```

**注意**: 
- SQL 必须是 SELECT 语句
- 不支持 WHERE 子句 (ABAP Open SQL 限制)
- 使用 `UP TO n ROWS` 限制结果集

### 3. ddic 端点 (表名查询)

**端点**: `POST /sap/bc/adt/datapreview/ddic`

**请求头**:
- `Accept: application/vnd.sap.adt.datapreview.table.v1+xml`
- `X-CSRF-Token: <token>`

**URL 参数**:
- `ddicEntityName` - 表名
- `rowNumber` - 返回行数

**示例**:
```
POST /sap/bc/adt/datapreview/ddic?ddicEntityName=MARA&rowNumber=10
```

## 结论

Data Preview API 已成功验证, 可用于:

1. **表数据查询** - 直接获取表内容
2. **数据预览** - 替代 SE16/SE16N 部分功能
3. **自动化测试** - 通过 API 验证数据

- 不支持 WHERE 条件 (ABAP Open SQL 限制)
- 需要 CSRF Token 认证
- 返回格式为 XML (非 JSON)
