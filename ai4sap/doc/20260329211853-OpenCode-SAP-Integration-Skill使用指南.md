## 配置说明

### SAP 连接配置

在项目目录创建 `config/sap_config.yaml`：

```yaml
SAP_CONFIG:
  USER: your_user
  PASSWD: your_password
  CLIENT: "400"
  LANG: ZH

ADT_CONFIG:
  BASE_URL: http://your-sap-host:port
```

### 配置管理原则

- ✅ 使用配置文件
- ✅ 添加 .gitignore 排除 secrets
- ✅ dev/staging/prod 环境隔离

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 401 认证失败 | 检查用户名密码，确认 Basic Auth |
| 404 对象不存在 | 确认表名/程序名大小写 |
| 响应慢 | 减少数据量，添加分页/缓存 |
| 内存溢出 | 使用分页获取，流式处理 |
