# SAP动态SQL查询REST API

## 概述

提供REST API接口，支持传递SQL语句给SAP系统并返回查询结果。

## 架构

```
客户端 -> FastAPI/Python -> SAP ICF Handler (ZZSQL) -> SAP数据库
```

## SAP端点 (需部署)

### ABAP程序: ZZSQ_SQL_QUERY

**ICF路径**: `/sap/bc/zzsql`

**参数**:
| 参数 | 必填 | 说明 |
|------|------|------|
| sql | 是 | SQL查询语句 |
| maxrows | 否 | 最大返回行数，默认100 |

**使用示例**:
```bash
curl -u U1170:hysoft888999 \
  "http://mysap.goodsap.cn:50400/sap/bc/zzsql?sap-client=400&sql=SELECT+*+FROM+mara&maxrows=10"
```

### 安全限制

- 仅支持 SELECT 查询
- 禁止 DROP/DELETE/UPDATE/INSERT/ALTER/CREATE
- 最大返回1000行

## Python FastAPI端点

### 1. POST /api/sap/query

执行原始SQL语句。

**请求**:
```json
{
  "sql": "SELECT * FROM mara WHERE mtart = 'HALB'",
  "max_rows": 10
}
```

**响应**:
```json
{
  "success": true,
  "data": [...],
  "row_count": 10,
  "execution_time_ms": 125.5
}
```

### 2. POST /api/sap/table

参数化查询表。

**请求**:
```json
{
  "table": "MARA",
  "fields": "MATNR,MTART,MBRSH",
  "where": "MTART = 'HALB'",
  "order_by": "MATNR",
  "limit": 10,
  "offset": 0
}
```

### 3. GET /api/sap/health

健康检查。

## 部署步骤

### 1. 部署ABAP程序到SAP

1. 复制 `code/zzsql.abap` 到SAP
2. 创建ICF节点: `/sap/bc/zzsql`
3. 指向程序: ZZSQ_SQL_QUERY

### 2. 启动FastAPI服务

```bash
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 3. 环境变量

```bash
export SAP_BASE_URL=http://mysap.goodsap.cn:50400
export SAP_USER=U1170
export SAP_PASSWORD=hysoft888999
export SAP_CLIENT=400
```

## 测试示例

```bash
# 测试SAP端点
curl -u U1170:hysoft888999 \
  "http://mysap.goodsap.cn:50400/sap/bc/zzsql?sql=SELECT+*+FROM+mara+LIMIT+3"

# 测试FastAPI端点
curl -X POST http://localhost:8000/api/sap/query \
  -H "Content-Type: application/json" \
  -d '{"sql": "SELECT * FROM mara LIMIT 3", "max_rows": 3}'
```
