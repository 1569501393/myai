# SAP ADT REST API 参考文档

## 概述

本文档基于在 SAP 系统 `mysap.goodsap.cn:50400` (客户端400) 上的实际测试结果编写。

**系统类型**: SAP ECC / NetWeaver ABAP (非S/4HANA)

---

## 已验证可用的端点

### 1. 源码获取端点 (GET)

| 对象类型 | 端点格式 | 示例 |
|----------|----------|------|
| **程序** | `/sap/bc/adt/programs/programs/{name}/source/main` | `/sap/bc/adt/programs/programs/ZJQR000/source/main` |
| **Include** | `/sap/bc/adt/programs/includes/{name}/source/main` | `/sap/bc/adt/programs/includes/ZJQR0000_FRM/source/main` |
| **类** | `/sap/bc/adt/oo/classes/{name}/source/main` | `/sap/bc/adt/oo/classes/ZCL_TEST/source/main` |
| **表** | `/sap/bc/adt/ddic/tables/{name}/source/main` | `/sap/bc/adt/ddic/tables/MARA/source/main` |

**请求示例**:
```bash
curl -u "USER:PASSWORD" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/ZJQR000/source/main" \
  -H "Accept: text/plain" \
  -H "sap-client: 400"
```

**响应**: 源码文本 (text/plain)

## 需要替代方案的端点

### 表数据获取

| 端点 | 状态 | 替代方案 |
|------|------|----------|
| `/sap/bc/adt/ddic/tables/{name}/data` | ❌ 404 不存在 | 使用 `/sap/bc/zzjq?table={name}&rows={n}` |
| `/sap/bc/adt/datapreview/{name}` | ❌ 404 不存在 (S/4HANA专用) | 使用 `/sap/bc/zzjq` |

**替代端点示例**:
```bash
# 获取MARA表前3条数据
curl -u "USER:PASSWORD" \
  "http://mysap.goodsap.cn:50400/sap/bc/zzjq?table=MARA&rows=3" \
  -H "sap-client: 400"
```

## 常用请求头

| 头信息 | 值 | 说明 |
|--------|-----|------|
| `Accept` | `text/plain` | 用于源码获取 |
| `Accept` | `application/atom+xml` | 用于列表查询 |
| `Accept` | `application/vnd.sap.adt.*+xml` | ADT特定格式 |
| `sap-client` | `400` | SAP客户端号 |
| `sap-language` | `ZH` 或 `EN` | 登录语言 |
| `X-CSRF-Token` | `Fetch` | 获取CSRF令牌 (用于POST/PUT/DELETE) |

## 认证方式

### Basic Auth (当前使用)
```bash
curl -u "USER:PASSWORD" "http://host:port/..."
```

### Cookie Auth (SAP SSO)
```bash
curl -b "SAP_SESSIONID=..." "http://host:port/..."
```

---

## 错误码说明

| 状态码 | 说明 | 解决方案 |
|--------|------|----------|
| 200 | 成功 | - |
| 400 | 请求错误 | 检查参数格式 |
| 401 | 未认证 | 检查用户名密码 |
| 403 | 权限不足 | 联系管理员开通权限 |
| 404 | 端点不存在 | 端点不支持或对象不存在 |
| 406 | 不支持的内容类型 | 更换Accept头 |
| 500 | 服务器错误 | 检查SAP系统状态 |

## S/4HANA 专用端点 (不可用)

以下端点仅在 S/4HANA 系统上可用:

- `/sap/bc/adt/datapreview/{table}` - 数据预览
- `/sap/bc/adt/runtime/traces` - 性能跟踪
- `/sap/bc/adt/abapdoc/{type}/{name}` - ABAP文档

## Python 使用示例

```python
import requests
from requests.auth import HTTPBasicAuth

SAP_HOST = "mysap.goodsap.cn:50400"
AUTH = HTTPBasicAuth("U1170", "hysoft888999")
CLIENT = "400"

def get_source(object_type, object_name):
    """获取SAP对象源码"""
    endpoints = {
        "PROG": "/sap/bc/adt/programs/programs",
        "INCLUDE": "/sap/bc/adt/programs/includes",
        "CLASS": "/sap/bc/adt/oo/classes",
        "TABL": "/sap/bc/adt/ddic/tables",
    }
    
    endpoint = endpoints.get(object_type.upper())
    if not endpoint:
        return {"error": f"Unknown type: {object_type}"}
    
    url = f"http://{SAP_HOST}{endpoint}/{object_name}/source/main"
    
    response = requests.get(
        url,
        auth=AUTH,
        headers={
            "Accept": "text/plain",
            "sap-client": CLIENT
        }
    )
    
    if response.status_code == 200:
        return {"success": True, "source": response.text}
    else:
        return {"error": f"HTTP {response.status_code}"}

def get_table_data(table_name, rows=10):
    """获取表数据 (使用自定义端点)"""
    url = f"http://{SAP_HOST}/sap/bc/zzjq"
    
    response = requests.get(
        url,
        auth=AUTH,
        params={"table": table_name, "rows": rows},
        headers={"sap-client": CLIENT}
    )
    
    if response.status_code == 200:
        return {"success": True, "data": response.json()}
    else:
        return {"error": f"HTTP {response.status_code}"}
```
