## 六、错误排查

### 6.1 常见错误

| 错误代码 | 含义 | 解决方案 |
|----------|------|----------|
| 401 Unauthorized | 认证失败 | 检查用户名密码 |
| 403 X-CSRF-Token Required | CSRF Token缺失 | 先获取Token |
| 404 Resource Not Found | 对象不存在 | 检查对象名称 |
| 500 Internal Server Error | 服务器错误 | 检查服务配置 |
| ExceptionResourceNotAcceptable | Accept类型错误 | 更换Accept类型 |

### 6.2 CSRF Token 处理

```bash
# 获取Token
curl -c cookies.txt -u "USER:PASS" \
  "http://host/sap/bc/adt/discovery" \
  -H "x-csrf-token: Fetch" -D headers.txt

# 使用Token (POST请求)
CSRF=$(grep -i "^x-csrf-token" headers.txt | cut -d' ' -f2)
curl -b cookies.txt -X POST \
  "http://host/sap/bc/adt/activation" \
  -H "X-CSRF-Token: $CSRF" \
  -H "Content-Type: application/vnd.sap.adt.activation+xml"
```
