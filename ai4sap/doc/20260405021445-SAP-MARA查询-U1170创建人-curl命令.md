## 5. 注意事项

### 5.1 字段名称大小写

- SAP ADT freestyle 端点使用 **OpenSQL** 语法
- 表名和字段名在 SQL 中可以**小写**（系统会自动转换）
- 但字符串值（如 'U1170'）必须用**单引号**

### 5.2 认证方式

- **Basic Auth**: `-u "USER:PASSWORD"`
- 需要每次请求带上 Cookie（Session保持）
- CSRF Token 每次请求都需要获取新的

### 5.3 rowNumber 参数

- 控制返回的最大行数
- 默认 10 条
- 建议设置足够大的值（如 1000）以获取全部数据

### 5.4 Content-Type

- 必须使用 `Content-Type: text/plain`
- SQL 语句作为纯文本发送，不是 JSON
