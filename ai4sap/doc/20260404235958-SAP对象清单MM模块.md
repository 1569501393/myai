## 七、SAP 对象获取方法

### 7.1 程序源码

```bash
curl -u "USER:PASS" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/ZJQR000/source/main" \
  -H "Accept: text/plain" \
  -H "sap-client: 400"
```

### 7.2 类源码

```bash
curl -u "USER:PASS" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/oo/classes/ZCL_TEST/source/main" \
  -H "Accept: text/plain" \
  -H "sap-client: 400"
```

### 7.3 表数据

```bash
# 通过 zzjq 端点
curl -u "USER:PASS" \
  "http://mysap.goodsap.cn:50400/sap/bc/zzjq?table=MARA&rows=10" \
  -H "sap-client: 400"
```

### 7.4 表结构 (DDL)

```bash
curl -u "USER:PASS" \
```
