# SAP ADT REST API 技能文档

**版本**: 1.0  
**更新**: 2026-04-05  
**系统**: SAP S/4HANA @ mysap.goodsap.cn:50400

## 一、技能概述

本技能提供完整的 SAP ADT (ABAP Development Tools) REST API 集成解决方案，支持 SAP 开发对象源码的检索与分析。

### 1.1 适用场景

- 检索 SAP 程序源代码
- 获取 ABAP 类实现
- 分析 Include 程序依赖关系
- 提取文本符号和程序结构
- SQL 数据查询

### 1.2 环境配置

从 `.env` 文件读取配置：

```bash
# SAP 连接配置
SAP_HOST=mysap.goodsap.cn
SAP_PORT=50400
SAP_CLIENT=400
SAP_USER=U1170
SAP_PASS=hysoft888999

# ADT 配置
ADT_PROTOCOL=http
```

## 二、支持的 ADT API 端点

### 2.1 端点验证状态

| 分类 | 前缀 | 状态 | 说明 |
|------|------|------|------|
| **ADT Objects** | `/sap/bc/adt/oo/` | ✅ 可用 | 类、接口 |

| **Activation** | `/sap/bc/adt/activation/` | ⚠️ CSRF问题 | 对象激活 |

## 三、认证流程

### 3.1 获取 CSRF Token

所有 POST 请求都需要 CSRF Token：

```bash
# 获取 CSRF Token 和 Session Cookie
curl -c cookies.txt \
  -u "U1170:hysoft888999" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/discovery" \
  -H "x-csrf-token: Fetch" \
  -D headers.txt \
  -k > /dev/null

# 提取 CSRF Token
CSRF=$(grep -i "^x-csrf-token" headers.txt | cut -d' ' -f2 | tr -d '\r')
```

### 3.2 完整认证脚本

```bash
#!/bin/bash
# SAP ADT 认证脚本

HOST="mysap.goodsap.cn"
PORT="50400"
USER="U1170"
PASS="hysoft888999"

COOKIE_FILE="/tmp/sap_cookies.txt"
HEADER_FILE="/tmp/sap_headers.txt"

# 获取 CSRF Token
curl -s -c "$COOKIE_FILE" \
  -u "${USER}:${PASS}" \
  "http://${HOST}:${PORT}/sap/bc/adt/discovery" \
  -H "x-csrf-token: Fetch" \
  -D "$HEADER_FILE" -k > /dev/null

CSRF=$(grep -i "^x-csrf-token" "$HEADER_FILE" | cut -d' ' -f2 | tr -d '\r')
echo "CSRF Token: $CSRF"
```

## 四、核心 API 使用示例

### 4.1 类和接口 (/sap/bc/adt/oo/)

#### 获取类源代码

```bash
# 读取 ABAP 类源代码
curl -b cookies.txt \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/oo/classes/zcl_zosql_db_layer/source/main" \
  -H "Accept: text/plain"
```

**响应**:
```abap
class ZCL_ZOSQL_DB_LAYER definition
  public
  inheriting from ZCL_ZOSQL_DB_LAYER_BASE
  create public .

public section.

  methods ZIF_ZOSQL_DB_LAYER~COMMIT
    redefinition .
...
```

#### 获取类定义

```bash
# 读取类定义（局部类型）
curl -b cookies.txt \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/oo/classes/zcl_zosql_db_layer/source/definitions" \
  -H "Accept: text/plain"
```

#### 获取接口源代码

```bash
# 读取接口定义
curl -b cookies.txt \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/oo/interfaces/zif_zosql_db_layer/source/main" \
  -H "Accept: text/plain"
```

### 4.3 数据预览 (/sap/bc/adt/datapreview/)

#### SQL 自由查询

```bash
# 查询物料主数据
curl -b cookies.txt \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/datapreview/freestyle?rowNumber=100" \
  -H "Content-Type: text/plain" \
  -H "X-CSRF-Token: $CSRF" \
  -d "SELECT matnr, mtart, matkl, ernam, ersda FROM mara WHERE ernam = 'U1170'"
```

**响应** (XML格式):
```xml
<?xml version="1.0" encoding="utf-8"?>
<dataPreview:tableData xmlns:dataPreview="http://www.sap.com/adt/dataPreview">
  <dataPreview:totalRows>6</dataPreview:totalRows>
  <dataPreview:columns>...</dataPreview:columns>
</dataPreview:tableData>
```

#### 常用查询示例

```bash
# 查询用户创建的物料
-d "SELECT matnr, mtart FROM mara WHERE ernam = 'U1170'"

# 查询采购申请
-d "SELECT banfn, bnfpo, matnr, menge FROM eban WHERE ernam = 'U1170'"

# 查询会计凭证
-d "SELECT bukrs, belnr, gjahr, budat FROM bkpf WHERE usnam = 'U1170'"

# 查询登录记录
-d "SELECT bname, logon_date, logon_time, terminal FROM usr41 WHERE bname = 'U1170'"

# 查询程序列表
-d "SELECT name, subc, cnam, cdat FROM trdir WHERE cnam = 'U1170'"

# 查询文本符号
-d "SELECT prog, dynr, dtxt FROM d020t WHERE prog = 'ZJQR000'"

# 查询消息文本
-d "SELECT arbgb, msgnr, text FROM t100 WHERE arbgb = 'ZHY100' AND sprsl = '1'"
```

## 六、完整 curl 示例

### 6.1 一键查询脚本

```bash
#!/bin/bash
# SAP ADT 查询工具

HOST="mysap.goodsap.cn"
PORT="50400"
USER="U1170"
PASS="hysoft888999"

query_sap() {
  local sql="$1"
  local rows="${2:-100}"
  
  # 获取 CSRF Token
  curl -s -c /tmp/query_cookies.txt \
    -u "${USER}:${PASS}" \
    "http://${HOST}:${PORT}/sap/bc/adt/discovery" \
    -H "x-csrf-token: Fetch" \
    -D /tmp/query_headers.txt -k > /dev/null
  
  CSRF=$(grep -i "^x-csrf-token" /tmp/query_headers.txt | cut -d' ' -f2 | tr -d '\r')
  
  # 执行查询
  curl -s -b /tmp/query_cookies.txt \
    "http://${HOST}:${PORT}/sap/bc/adt/datapreview/freestyle?rowNumber=${rows}" \
    -H "Content-Type: text/plain" \
    -H "X-CSRF-Token: $CSRF" \
    -d "$sql"
}

# 使用示例
query_sap "SELECT matnr, mtart FROM mara WHERE ernam = 'U1170'"
```

### 6.2 批量获取源代码

```bash
#!/bin/bash
# 批量获取程序源代码

programs=("ZJQR000" "ZJQR001" "ZJQR002")

for prog in "${programs[@]}"; do
  echo "=== $prog ===" 
  curl -s -b cookies.txt \
    "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/${prog}/source/main" \
    -H "Accept: text/plain" > "${prog}.abap"
  echo "Saved to ${prog}.abap"
done
```

## 七、错误处理

### 7.1 常见错误

| 错误代码 | 含义 | 解决方案 |
|----------|------|----------|
| 401 Unauthorized | 认证失败 | 检查用户名密码 |
| 403 X-CSRF-Token Required | CSRF缺失 | 重新获取Token |
| 404 Resource Not Found | 对象不存在 | 检查对象名称 |
| 500 Internal Server Error | 服务器错误 | 检查SAP服务状态 |

### 7.2 CSRF Token 问题

```bash
# 如果遇到 CSRF 错误，重新获取
curl -c cookies.txt -u "USER:PASS" \
  "http://host/sap/bc/adt/discovery" \
  -H "x-csrf-token: Fetch" -D headers.txt -k > /dev/null

CSRF=$(grep -i "^x-csrf-token" headers.txt | cut -d' ' -f2)
```
