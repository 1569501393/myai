## 一、概述

SAP ABAP 程序中的文本符号 (Text Symbols) 和选择屏幕 (Selection Screen) 是用户界面的重要组成部分：

### 1.1 文本符号 (Text Symbols)
- 位置: 程序源码中 REPORT 语句之后
- 语法: `* Text-XXX: 描述文本`
- 用途: 定义程序标题、列标题、选择屏幕标签等

### 1.2 选择屏幕 (Selection Screen)
- 位置: 程序源码中 AT SELECTION-SCREEN 输出块之前
- 语法: `PARAMETERS` 和 `SELECT-OPTIONS`
- 用途: 定义用户输入界面

## 五、文本符号和选择屏幕的代码模式

### 5.1 文本符号模式

```
REPORT zprogram.
* Text-001: Program Title
* Text-002: Column Header 1
* Text-003: Column Header 2
* Text-004: Button Text
```

### 5.2 选择屏幕模式

```
SELECTION-SCREEN BEGIN OF SCREEN 100.

PARAMETERS: p_var1 TYPE char10,
            p_var2 TYPE char20.

SELECT-OPTIONS: s_date FOR sy-datum NO INTERVALS.

SELECTION-SCREEN END OF SCREEN 100.
```

### 5.3 从源码中提取文本符号

使用正则表达式从源码中提取：

```bash
# 提取文本符号
curl -s -u "U1170:hysoft888999" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/ZPROGRAM/source/main" -k | \
  grep -E "^\* Text-[0-9]+:" | \
  sed 's/^\* Text-\([0-9]\+\): \(.*\)/\1 \2/'
```

### 5.4 从源码中提取选择屏幕参数

```bash
# 提取 PARAMETERS
curl -s -u "U1170:hysoft888999" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/ZPROGRAM/source/main" -k | \
  grep -E "^\s*PARAMETERS" | \
  sed 's/PARAMETERS://g'

# 提取 SELECT-OPTIONS
curl -s -u "U1170:hysoft888999" \
  "http://mysap.goodsap.cn:50400/sap/bc/adt/programs/programs/ZPROGRAM/source/main" -k | \
  grep -E "^\s*SELECT-OPTIONS" | \
  sed 's/SELECT-OPTIONS://g'
```

## 六、完整 curl 测试脚本

```bash
#!/bin/bash
# 获取程序文本符号和选择屏幕

HOST="mysap.goodsap.cn"
PORT="50400"
USER="U1170"
PASS="hysoft888999"
PROGRAM="ZJQR0000"

echo "=== 获取程序 ${PROGRAM} 文本符号和选择屏幕 ==="
echo ""

# 1. 获取程序源码
echo "1. 获取程序源码..."
curl -s -u "${USER}:${PASS}" \
  "http://${HOST}:${PORT}/sap/bc/adt/programs/programs/${PROGRAM}/source/main" -k > /tmp/${PROGRAM}_source.txt

# 2. 获取程序元数据
echo "2. 获取程序元数据..."
curl -s -u "${USER}:${PASS}" \
  "http://${HOST}:${PORT}/sap/bc/adt/programs/programs/${PROGRAM}" -k > /tmp/${PROGRAM}_meta.xml

# 3. 获取对象结构
echo "3. 获取对象结构..."
curl -s -u "${USER}:${PASS}" \
  "http://${HOST}:${PORT}/sap/bc/adt/programs/programs/${PROGRAM,,}/objectstructure" -k > /tmp/${PROGRAM}_structure.xml

# 4. 提取文本符号
echo ""
echo "=== 文本符号 ==="
grep -E "^\* Text-[0-9]+:" /tmp/${PROGRAM}_source.txt || echo "无文本符号"

# 5. 提取选择屏幕参数
echo ""
echo "=== 选择屏幕参数 (PARAMETERS) ==="
grep -E "^\s*PARAMETERS" /tmp/${PROGRAM}_source.txt || echo "无 PARAMETERS"

# 6. 提取选择选项
echo ""
echo "=== 选择屏幕选项 (SELECT-OPTIONS) ==="
grep -E "^\s*SELECT-OPTIONS" /tmp/${PROGRAM}_source.txt || echo "无 SELECT-OPTIONS"

# 7. 提取 FORM 例程
echo ""
echo "=== FORM 例程 ==="
grep -E "^\s*FORM\s+[a-zA-Z0-9_]+" /tmp/${PROGRAM}_source.txt | sed 's/FORM //g' | sed 's/ .*//g'

echo ""
echo "=== 完成 ==="
```

## 七、相关 ADT REST API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/sap/bc/adt/programs/programs/{name}/source/main` | GET | 获取程序源码 |
| `/sap/bc/adt/programs/programs/{name}` | GET | 获取程序元数据 |
| `/sap/bc/adt/programs/programs/{name}/objectstructure` | GET | 获取程序对象结构 |
| `/sap/bc/adt/programs/includes/{name}/source/main` | GET | 获取 Include 源码 |
| `/sap/bc/adt/vit/wb/object_type/progpx/object_name/{name}` | GET | 获取文本元素元数据 |
| `/sap/bc/adt/functions/groups/{name}/source/main` | GET | 获取函数组源码 |

---

## 八、注意事项

1. **文本符号不是通过独立 API 获取**: 需要从程序源码中解析提取
2. **选择屏幕定义在源码中**: PARAMETERS 和 SELECT-OPTIONS 语句定义了选择屏幕
3. **文本元素对象 (PROG/PX)**: 存储文本符号、选择屏幕文本等
4. **SAP GUI 协议**: 某些文本元素需要通过 `application/vnd.sap.sapgui` 类型访问
